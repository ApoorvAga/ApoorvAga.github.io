Context Menu

I designed a general purpose context-menu for the company's website. It had to work with over 5 drastically different components without minimal change.

Earlier, the user had to go into one of the resources to make changes.
To make things easier and faster, I designed a context menu where user can simply right-click the particular resource and make changes to it.

Technologies:
  • - Vue.js
  • - Javascript
  • - HTML/CSS
Sub-Menus

Sub-Menus was the most challenging part of the project.
I had to make sure the component wouldn't behave unexpectedly as it had several sub-menus with a lot of data.

It also supported multiselect resources like linking labels and assigning users. Not just simple buttons.
