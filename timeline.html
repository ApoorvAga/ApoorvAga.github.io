Timeline Component

Timeline Component keeps track of all the actions performed on the website by the user in a clean and systematic way.

Whenever an action is performed on the website, the timeline service is called which saves the resource and the difference between the data to the db.
In addition to the common Timeline Component, each major resource on the website has it's own Timeline Component.

Technologies:
  • - Sails.js
  • - MongoDB
  • - RabbitMQ
  • - Vue.js
  • - Javascript
  • - HTML/CSS
More Details

User can view more details by clicking of a specific timeline event. It contains the details of the action performed, the resource and many more.

The most challenging part was creating timeline events for json fields.
Json fields could contain any type of fields like list, test, date, numbers, array or an array of json fields. For this the details view shows a table with the field name, old value and new value.
